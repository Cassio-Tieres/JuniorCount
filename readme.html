# Nascimento O fruto deste projeto veio de uma necessidade pessoal de organizar os gastos que estávamos (minha família e eu) tendo em um determinado estabelecimento. # Tecnologias As principais tecnologias utilizadas foram: * o WordPress; * PHP; * Bootstrap; * HTML 5; * CSS3. por ser um projeto extremamente simples, que não necessitaria de muita interatividade, o JavaScript acabou ficando de fora. # Plugins Alguns plugins foram de extrema necessidade para que esse projeto funcionasse bem, e os principais são: * Contact Form 7; * CF7 Google Sheets Connector; * Classic Editor; * Easy WP SMTP; * SG Security; Também foram aplicados alguns outros plugins que ainda não foram utilizados, mas que poderão ser futuramente: * Advenced Custom Field PRO; * Custom Post Type;
